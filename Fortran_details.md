    * Declaring variables and arrays
    * Arithmetic operations
    * DO and DO WHILE loops
    * IF statements
    * Functions

* INTENT statements
* OPTIONAL arguments
* INTERFACES
* MODULES

* Derived types
* Defining and overloading operators
* Allocating and deallocating dynamic memory
* POINTERs and ALLOCATABLE arrays
